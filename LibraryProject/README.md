LibraryProject Backend

This is the foundational setup for a Django backend application designed to manage library resources (books, authors, transactions).

Setup Instructions

Clone the repository: git clone [Your Repository URL]

Navigate to the project directory: cd LibraryProject

Create and activate virtual environment:

python3 -m venv venv

source venv/bin/activate (Linux/macOS)

Install dependencies: pip install django

Run development server: python manage.py runserver
